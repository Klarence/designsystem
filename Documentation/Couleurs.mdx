# Couleur d'arrière-plan

Le blanc est la couleur d'arrière-plan par défaut pour les pages, garantissant une lisibilité optimale du texte et des éléments.

Note : Les ratios de contraste sont calculés sur un fond blanc. Modifier l'arrière-plan peut compromettre l'accessibilité de plusieurs composants. Maintenir le fond blanc par défaut est fortement recommandé.

<ColorItem title="Blanc" subtitle="rgb(255, 255, 255)" colors={['#FFFFFF']} />


# Couleur du texte

La couleur par défaut pour la plupart des éléments textuels sur les pages, tels que les paragraphes et les titres.

# Palette principale
La palette principale comprend 2 couleurs de base : une couleur principale et une couleur d'accent.

Celles-ci doivent être prédominantes dans les interfaces.

## Couleur principale
La couleur principale est un groupe de 6 couleurs, de la plus pâle à la plus foncée. Cela donne aux concepteurs et développeurs web beaucoup de flexibilité lors de la création des interfaces.
